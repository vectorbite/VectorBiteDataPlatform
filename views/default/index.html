Welcome!

Welcome to VectorBiTE's new Eco-informatics Data Platform. Please register and take a look around! We are in the final stages of developing the first database named VecDyn, which is a global database for spatially- and temporally- explicit presence-absence and population abundance data. We will accept and distribute data for both animal and plant disease vectors. For further information please visit the VecDyn database guidelines on {{=A('GitHub', _href='https://github.com/vectorbite/VectorBiteDataPlatform/blob/master/static/Documentation/VecDyn/VecDynUserGuide.md')}}. Pages on our GitHub site provide details and instructions on accessing and submitting data. You can also access data collection templates and examples that will help you prepare data for submission. The GitHub page also hosts the development code for the database and web application which has been built using {{=A('Web2py (v2.16.1)', _href='http://www.web2py.com/')}} , a python based web development framework.
