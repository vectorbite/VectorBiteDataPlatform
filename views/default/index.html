Welcome!

Welcome to VectorByte's new Eco-informatics Data Platform. We are in the final stages of developing the two database named VecDyn and Vectraits, which is a global database for spatially- and temporally- explicit presence-absence and abundance data. We will be up and running around January 2018, from then you'll be able to download and submit data to the database. We will accept and distribute data for both animal and plant disease vectors. The GitHub page also hosts the development code for the database and web application which has been built using {{=A('Web2py (v2.16.1)', _href='http://www.web2py.com/')}} , a python based web development framework.
