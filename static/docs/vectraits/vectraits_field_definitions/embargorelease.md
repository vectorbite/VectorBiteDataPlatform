# embargorelease

## Type
Date

## Description
**This is a Required Field!**

The date at which the emargo on the data is lifted (in **DD/MM/YYYY**). If there is no embargo, set this to a time in the past (such as 01/01/1970).
## Example
*01/01/1992*
